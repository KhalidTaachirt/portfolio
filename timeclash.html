About Time Clash


Status: Finished
Tools used: Unity, C#, PHP, MySQL, Visual Studio and Git
Roles: Game Programmer and Project Manager

During the making of Time Clash, I worked with a team of 4. I primarily worked on the player and enemy code. Since the game is a bullet hell a lot of bullets were needed, which is why I also implemented the object pooling design pattern to improve the performance of the game. Besides working on the game I also made use of GitLab CI/CD to automatically make a WebGL build and upload it to itch.io.

My Contributions:

  • Bullet object pool
  • Enemy AI
  • Time Jump (Dash) ability
  • Player movement and shooting
  • Automatic WebGL build and upload to itch.io with GitLab CI/CD
  • Game metrics using PHP and MySQL with error handling

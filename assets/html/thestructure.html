About The Structure


Status: In development
Tools used: Unity, C#, Visual Studio and Git
Roles: Game Programmer, Game Designer and Project Manager

I am currently working on The Structure with a team of 5. As a game programmer, I implemented multiple systems in the game based on the designs that the team and I made. Throughout the project, I helped the team in numerous ways like making extensive user stories, controlling project scope, helping teammates solve problems related to coding and solving merge conflicts.

My Contributions:

  • Boss finite state machine including animations
  • Damage falloff and numbers
  • Collisions with objects
  • Player spherecasting
  • Improved the camera
  • Gameplay design

Boss finite state machine

I wrote a finite state machine for the boss which includes the various states that the boss has. While writing the state machine in C# I made sure to write it so more states could be added later on. This state machine controls the several animations and transitions in the animator state machine and provides one access point for the boss state to be controlled.

State machine code on GitHub



Damage fallof and numbers

I developed a system which changes the damage that the player does based on the distance between the player and enemy. When a bullet hits an enemy a number is shown displaying the damage that the bullet did, based on the damage the color also changes.

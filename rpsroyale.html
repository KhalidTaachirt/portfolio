About RPS Royale


Status: Finished
Tools used: JavaScript, P5js, PHP, MySQL, Visual Studio Code and Git

While making RPS Royale, I worked in a team of 4 where I primarily wrote the backend of the game. During this project, I wrote the entire backend which resulted in a fully functional lobby system. This is the first time that I made a multiplayer game and where my creativity shined a lot since the entire system was made with PHP and JavaScript. Besides writing the backend of the game I also made a game portal for all the games that the students made including a thumbnail, description and genre.

My Contributions:

  • Multiplayer compatibility
  • Create and join lobby system
  • Game Portal which showcases student games
  • Made the game playable on both mobile and pc
